# Proyecto base 
# vue 3 - vue-router 4 - pinia

### Archivo .env

| Variable de entorno     | Valor                      |
|-------------------------|----------------------------|
| VITE_APP_BASE_URL_REST  | URL Proyecto Firebase      |
| VITE_APP_BASE_URL_LOGIN | URL API Rest Auth Firebase |
| VITE_APP_BASE_URL_KEY   | API KEY Proyecto Firebase  |

### Uso

  * Clono el repo
  * npm install
  * npm run dev
